AI-powered fire detection using computer vision. Upload images and get instant fire detection with red bounding boxes.

## 📌 Features

- 🖼️ Upload images for fire detection
- 📷 Webcam support for real-time detection
- 🎨 Color-based fallback detection
- 🔴 Visual red box highlighting
- ⚙️ Adjustable sensitivity slider
